# 06-01 Template Library page

## Цель

Публичная страница `/templates` с поиском по названию, фильтром по движку, секцией пресетов и возможностью загрузить шаблон в sandbox или клонировать к себе.

## Зачем

Точка открытия для новых пользователей: можно сразу попробовать готовый шаблон без регистрации. Поиск и фильтр помогают найти нужный пример среди публичных шаблонов. Пресеты хранятся на бэкенде.

## Use Cases

- Гость открывает `/templates` — видит секцию "Quick Start" с пресетами для каждого движка и список публичных шаблонов
- Пользователь набирает "hello" в поиске — список фильтруется с debounce 300ms
- Пользователь выбирает фильтр "Pug" — показываются только pug-шаблоны
- Пользователь нажимает "Use in Sandbox" — шаблон загружается в слот A, редирект на `/sandbox`
- Авторизованный нажимает "Clone" — шаблон появляется в My Templates, snackbar подтверждение
- Гость нажимает "Clone" — snackbar "Sign in to clone templates"

## Критерии готовности

- [ ] Debounced-поиск 300ms перезагружает список
- [ ] Фильтр по движку работает (chips или select)
- [ ] Quick Start пресеты: handlebars, pug, ejs — хардкод на фронте
- [ ] "Use in Sandbox" → sandbox store + navigate `/sandbox`
- [ ] "Clone" для auth → клонирует, snackbar; для гостя → snackbar с предложением войти
- [ ] Skeleton loading во время загрузки
