# 06-02 "Save as Template" из Sandbox

## Цель

Диалог сохранения текущего слота как шаблона, доступен только авторизованным пользователям.

## Use Cases

- Авторизованный пользователь написал шаблон — нажимает "Save as Template", вводит название, выбирает публичный/приватный — шаблон появляется в My Templates
- Диалог показывает движок активного слота (read-only) — пользователь не путается
- Гость видит кнопку "Save as Template" серой или скрытой — подсказка "Sign in to save"
- После сохранения — диалог закрывается, snackbar "Template saved!"

## Критерии готовности

- [ ] Кнопка "Save as Template" в action bar sandbox, доступна только авторизованным
- [ ] Диалог: поле "Name" (обязательное), "Description" (опционально), toggle "Public"
- [ ] Движок активного слота показывается в диалоге (не редактируется)
- [ ] После успешного создания — snackbar подтверждение, диалог закрывается
- [ ] Шаблон появляется в My Templates без перезагрузки страницы
