# 00-02 TypeScript типы

## Цель

Определить все shared TS-интерфейсы в одном файле `src/types/index.ts`.

## Зачем

Единый источник типов исключает дублирование и рассинхронизацию между модулями. Все сущности — User, Engine, Slot, SandboxState, Template, RenderRun, BenchmarkResult — описаны в одном месте.

## Use Cases

- Разработчик пишет store или API-модуль — импортирует тип из `@/types` без поиска по кодовой базе
- TypeScript ловит несоответствие типов на этапе компиляции

## Критерии готовности

- [ ] Файл `src/types/index.ts` создан, все основные сущности экспортированы
- [ ] `npm run type-check` проходит без ошибок
