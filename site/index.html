Что делает проект

Пользователь редактирует шаблоны и общий JSON-контекст, запускает preview или benchmark, сравнивает результат двух slot-ов и сохраняет удачные шаблоны или запуски для дальнейшего анализа.

Возможности

  • Два template slot-а для сравнения вариантов.
  • Общий JSON-контекст для честного сравнения движков.
  • Preview и benchmark с avg, min, max, p95 и размером output.
  • Публичная библиотека шаблонов и личный dashboard.
  • Share-ссылки на сохранённое состояние sandbox.

Поддерживаемые движки

Движки загружаются на клиенте по требованию, а редактор подстраивает подсветку под активный slot.

Handlebars Pug EJS

Стек

Клиентская песочница, редактор кода и backend-слой для сохранения результатов экспериментов.
